# 浏览器渲染原理

```html
<html>
  <head>
    <title>Page Title</title>
    <link rel="stylesheet" href="style.css" />
    <script src="script.js"></script>
  </head>
  <body>
    <h1>Title</h1>
    <div>
      <p><strong>Lorem ipsum dolor</strong> sit amet.</p>
    </div>
  </body>
</html>
```

```mermaid
stateDiagram-v2
classDef root font-family:monospace
classDef treeroot fill:#F8CECC,stroke:#C36B68,font-weight:bold
classDef dom fill:#DAE8FC,stroke:#89A5CD
classDef text fill:#FFF2CC,stroke:#DDC06D
    document:::treeroot --> head:::dom
    document --> body:::dom
    head --> title:::dom
    t1:::text: "Page Title"
    title --> t1
    head --> link:::dom
    head --> script:::dom
    body --> h1:::dom
    t2:::text: "Title"
    h1 --> t2
    body --> div:::dom
    div --> p:::dom
    p --> strong:::dom
    t3:::text: "Lorem ipsum dolor"
    strong --> t3
    t4:::text: "sit amet."
    p --> t4
```

```mermaid
stateDiagram-v2

    div --> p:::dom

    strong --> t3
    t4:::text: "sit amet."
    a4:::anom: (匿名行盒)
    p --> a4
    a4 --> t4
```
