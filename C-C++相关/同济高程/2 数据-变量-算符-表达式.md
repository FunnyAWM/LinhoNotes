:::: details 错误示范

::: code-group

```cpp [大括号不换行]
// [!code escape-format]
#include<iostream>
using namespace std;
int main() {
    cout << "Hello, world." << endl;
    return 0;
}
```

```cpp [压行]
// [!code escape-format]
#include<iostream>
using namespace std;
int main()
{ cout << "Hello, world." << endl; return 0; }
```

```cpp [过度换行]
// [!code escape-format]
#include<iostream>
using namespace std;
int main()
{
    cout
      <<
      "Hello, world."
      << endl;
    return 0;
}
```

```cpp [使用 “\” 折行]
// [!code escape-format]
#include<iostream>
using namespace std;
int main()
{
    cout  \
      <<  \
      "Hello, world."  \
      << endl;
    return 0;
}
```
