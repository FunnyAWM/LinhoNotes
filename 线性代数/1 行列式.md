# 1.1 二阶与三阶行列式

对于方程组
$$
\left\{ \matrix{
  3{x_1} - 2{x_2} = 12 \hfill \cr 
  2{x_1} + {x_2} = 1 \hfill \cr}  \right.
$$
将其系数写成
$$
D = \left| {\matrix{
   3 & { - 2}  \cr 
   2 & 1  \cr 

 } } \right|,{D_1} = \left| {\matrix{
   {12} & { - 2}  \cr 
   1 & 1  \cr 

 } } \right|,{D_3} = \left| {\matrix{
   {12} & 3  \cr 
   1 & 2  \cr 

 } } \right|
$$
